What is Elecration ?

Elecration is fast, high-performance and user-friendly with modernized codes. .

Some system about Elecration

OPC (Operator Control): Operator management and list controller system.
SVC (Server Control): Server control and event changer system.
PCC (Plugin Command Control): Plugin control and event changer system.
DCM (Discord Messenger): Get any information about your server from discord.

ElecrationLCC+

As an extra, you can use the electration fork by integrating it with the license system, for this you need to contact mixeration directly.

How it works: The license system is managed directly with Python or PhPMyAdmin. The required MySql is purchased by the user and integrated into ElecrationLcc+.
